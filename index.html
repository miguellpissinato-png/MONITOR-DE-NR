    Sobre o Sistema

    O Monitor de NR é um sistema automatizado de vigilância das Normas Regulamentadoras do Ministério do Trabalho e Emprego (MTE). Ele acessa diariamente o portal gov.br e verifica se houve qualquer alteração no conteúdo das NR vigentes.

    Como funciona

    1. Todo dia às 07:00 (horário de Brasília) o sistema acessa o portal do MTE automaticamente.

    2. Compara o conteúdo atual de cada NR com a versão da última verificação.

    3. Se detectar diferença, registra a alteração com data, nome da NR e link direto.

    4. Alterações permanecem visíveis na aba Recentes por 7 dias e depois são movidas para o Histórico permanente.

    Fontes Monitoradas

    • Gov.br / MTE — Página oficial de cada Norma Regulamentadora

    • Página índice — Portal de NR do Ministério do Trabalho

    Sistema hospedado gratuitamente no GitHub Pages. Automatização via GitHub Actions. Nenhum servidor pago é necessário.
